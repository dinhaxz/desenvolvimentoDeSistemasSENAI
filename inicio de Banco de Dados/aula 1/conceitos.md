## Gerenciamento de Banco de Dados

Conjunto de práticas para organizar, proteger, otimizar e administrar bancos de dados. Envolve controle de usuários, backups, desempenho e integridade das informações.

## Bancos de Dados Relacionais

Armazenam dados em tabelas com linhas e colunas, relacionadas por chaves primárias e estrangeiras. Usam SQL e seguem regras de integridade para manter os dados consistentes.

## Diagrama de Dados (Diagrama ER)

Representação visual das entidades, atributos e relacionamentos de um banco. Serve para planejar e entender a estrutura do banco de dados.
